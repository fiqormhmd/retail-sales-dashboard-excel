# 📊 Retail Sales Dashboard (Excel)

## 📌 Project Overview

This project analyzes retail sales data using Microsoft Excel to identify sales trends, top-performing products, categories, regions, and states. The dashboard is interactive using Pivot Tables, Pivot Charts, and Slicers.

---

## 🎯 Objectives

- Clean retail sales data
- Perform Exploratory Data Analysis (EDA)
- Build Pivot Table analysis
- Create an interactive Excel Dashboard
- Generate business insights for decision making

---

## 🛠 Tools

- Microsoft Excel
- Pivot Table
- Pivot Chart
- Slicer
